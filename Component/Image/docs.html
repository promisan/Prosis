Output: suppressed

Output: suppressed
Parameters:
   imageObj: any, optional, imageObj
   backgroundColor: string, optional, backgroundColor
   mirrorHeight: numeric, optional, mirrorHeight - Number of pixels high the mirror will be, this will increase the overall image height by this amount.
   startOpacity: numeric, optional, startOpacity - must be between 0 and 100, controls the opacity at the top of the mirror
